Craftics

Turn-based tactical RPG combat inside Minecraft.

Fight through 18 biomes across the Overworld, Nether, and End in grid-based combat using your actual inventory, weapons, armor, and potions. Every item has a purpose in battle.

⬇ Download Read the Docs
Minecraft 1.21.1 Fabric Loader ≥0.16 Java 21+ Fabric API

Features

Everything vanilla. Nothing wasted.

⚔

Combat System

3 AP per turn. 16 weapon types with unique abilities. Sweep, stun, pierce, smash. Percentage-based defense. 40+ usable items.

🧠

40+ Enemy AIs

Every mob thinks differently. Skeletons kite. Creepers fuse. Endermen teleport. Spiders pounce. Bosses have phase transitions.

🗺

18 Biomes

Branching paths across Overworld, Nether, and End. 5 levels per biome. 90 levels total. Randomized per world.

🛡

Armor Trim System

18 trim patterns with per-piece bonuses plus 10 trim materials. Match all 4 for a full-set bonus. 7 armor classes from Brawler to Juggernaut.

💰

Trader Economy

8 trader types between levels. Weaponsmiths, alchemists, decorators, and more. Spend emeralds earned from victories.

🔧

Fully Moddable

Custom arena maps, JSON biome datapacks, Java API for custom AI. Hot reload with /reload.

Three Dimensions

Progress through a branching path of increasing danger.

Overworld

8 biomes · branching paths · Warden boss

Plains through Dark Forest, branching to Desert or Jungle. Converges at Underground Caverns and culminates in The Deep Dark, home of the Warden.

The Nether

5 biomes · linear · fire & lava

Nether Wastes through Soul Sand Valley, Crimson Forest, Warped Forest, to Basalt Deltas. Fire, lava, and the most dangerous hostile mobs.

The End

4 biomes · final act · Ender Dragon

Outer End Islands, End City, Chorus Grove, and Dragon's Nest, the final confrontation with the Ender Dragon.

No new items. No new blocks. (Except the Level Select.) Craftics gives combat meaning to everything already in the game: swords, axes, bows, potions, food, armor trims, goat horns, and more.
